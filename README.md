## Installation

The preferred way to install this yii2-extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require "ru-tld/hidev-config"
```

or add

```json
"ru-tld/hidev-config": "*"
```

to the require section of your composer.json.
